<h1>Aloha</h1>
<p>Aloha is MERN stack application with some social media's main features</p>

### Techstack
- Nodejs
- Mongo
- Socket Io
- AWS: S3

### Requirements
- Node >= 16, yarn
- Docker, docker-compose
- Make

### Development Setup
1. Setup development infrastructure
```
make bootstrap
```

2. Start app
```
yarn dev
```

3. Server runs on `https://localhost:5000`
